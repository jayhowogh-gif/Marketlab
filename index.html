기호 · 색 읽는 법

색으로 읽기 (한국식)
빨강 — 상승 · 양(+)의 상관 · 순풍. 오르거나 같은 방향으로 움직여요.
파랑 — 하락 · 음(−)의 상관 · 역풍. 내리거나 반대 방향으로 움직여요.
지표 관계 그래프 (탐색 탭)
선 굵기 = 관계 강도. 굵을수록 서로 강하게 밀고 당겨요.
실선(빨강) = 양의 상관. A가 오르면 B도 오르는 관계.
점선(파랑) = 음의 상관. A가 오르면 B는 내리는 관계.
노드(원) 크기 = 변동성·중요도. 클수록 더 크게 흔들리는 지표.
! (주황 원) = 평소와 달라진 관계. 나만의 기준값을 벗어나 이탈한 연결이에요.
신호등 ● (밸류에이션 · 익절 판정)
초록 = 좋음. 저평가 매력 / 상승 여력 구간.
주황 = 보통·중립. 대체로 적정, 뚜렷한 신호 없음.
빨강 = 주의. 고평가 주의 / 익절 고려 구간.
